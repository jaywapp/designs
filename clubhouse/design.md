# Design Spec — Clubhouse (clubhouse)

FC 마포 유나이티드 축구 클럽 랜딩. **친근한 클럽하우스** 미학의 라운드 카드 레이아웃.
출처: `clubhouse.html`

## 1. 디자인 컨셉

- **테마**: 라이트 모드, 크게 둥근(soft) 카드 기반의 따뜻하고 친근한 톤
- **분위기**: 커뮤니티 클럽하우스 — 부드러운 그림자, 라운드 코너, 라임 포인트
- **밀도**: 중밀도, 카드 그리드 대시보드
- **최대 폭**: `1600px` 중앙 정렬
- **인터랙션**: 정적(hover 스타일 최소), 접근성 focus-visible 처리

## 3. 타이포그래피

- **폰트**: `"Arial Narrow","Pretendard","Noto Sans KR",Arial,sans-serif` + `-webkit-font-smoothing:antialiased`
- **헤드라인 H1**: `67px` / `font-weight:950` / `line-height:.99` / `letter-spacing:-4px`
- **날짜**: `48px` / `letter-spacing:-3px`
- **스코어**: 라임 배경 알약, `950 27px`
- **라벨**: `900`, 넓은 자간(`1.3~2.2px`), 대문자

---

## 4. 레이아웃 구조

### 골격
- `topline`(8px 컬러바) → `header`(102px) → `main`(max 1600px, 중앙)

### 히어로 (`.hero`, `1.35fr : 0.65fr(min 410px)`)
- **좌(intro)**: 네이비 라운드(36px) 히어로 — 라임 아이브로우 → H1 → lead → 라임/고스트 버튼. 원형 장식(`:before` 두꺼운 라임 링, `:after` 회전 타원)
- **우(match)**: 흰 카드(NEXT MATCH) — 라이브 점, 날짜, 대진(mini-crest our=네이비/their=라이트), 하단 foot

### 클럽하우스 그리드 (`.grid`, `0.88 : 1.2 : 0.92`)
- **result**: 그린-다크 카드, 라임 스코어, 득점자
- **dashboard**: RSVP + 포지션 현황(진행 바)
- **players**: 포지션 필터 pill + 선수 카드(CSS로 그린 초상 실루엣)

---

## 5. 컴포넌트

### 크레스트 (`.crest`)
- 라운드 방패(`18px 18px 24px 24px`), 네이비 배경 + `inset 0 -7px 0 그린`(하단 그린 밑단)

### 미니 크레스트 (`.mini-crest`)
- our: 네이비+그린 밑단 / their: 라이트+테두리

### RSVP (`.rsvp`)
- 3버튼 그리드, 참석=그린 채움. `faces`: 겹치는 원형 아바타 스택(+9)

### 포지션 진행 바
- 인라인 스타일 바: 공격수 75%(그린) / 미드필더 100%(네이비)

### 선수 카드 (`.portrait`)
- 이미지 없이 CSS만으로 초상 실루엣(원형 얼굴 + 어깨 반원)

### 버튼 (`.btn`)
- primary: 라임 배경 네이비 텍스트 / ghost: 반투명 흰 테두리

---

## 6. 반응형

- `@media(max-width:900px)`:
  - 네비 링크 숨김(로그인만), 히어로/그리드 1열
  - intro 패딩 축소, H1 `67px → 48px`
