## 1. 디자인 컨셉

- **테마**: 라이트 모드, 종이(paper)/크림 배경의 편집 디자인
- **분위기**: 데일리 프로덕션 브리프 — 대형 세리프 헤드라인, 여백 중심, 신문/잡지 톤
- **밀도**: 저·중밀도 (읽기 편한 브리핑 문서 레이아웃)
- **최소 폭**: `1200px`
- **인터랙션**: 상태 카드 hover 시 살짝 떠오름(`translateY(-3px)`)

---

## 2. 컬러 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--paper` | `#f3efe6` | 페이지 배경 (웜 페이퍼) |
| `--ink` | `#1d231d` | 기본 텍스트 (잉크 블랙-그린) |
| `--muted` | `#6b6d61` | 보조 텍스트 |
| `--line` | `#d5d0c3` | 경계선 (연한 베이지) |
| `--orange` | `#e8682e` | 주 액센트 (강조·경고·D-day) |
| `--olive` | `#79834a` | 보조 액센트 (정상·완료) |
| `--cream` | `#fbf8f1` | 카드 hover 배경 |

**상태 색 규칙**: 정상/완료=올리브, 강조/경고/현재=오렌지.

---

## 3. 타이포그래피

- **본문**: `"Segoe UI","Noto Sans KR",Arial,sans-serif`
- **헤드라인**: `Georgia,"Times New Roman",serif` — **세리프**가 핵심 (에디토리얼 톤)
  - H1: `58px` / `font-weight:500` / `line-height:.98` / `letter-spacing:-.045em`
  - D-day 카운트(`.count strong`): Georgia `132px` / weight 400 — 거대 타이포
  - 섹션 타이틀 H2: Georgia `27px`
- **모노스페이스**: `Consolas` (stream, 메타, 날짜)
- **아이브로우**: `11px` / 대문자 / `letter-spacing:.18em` / 오렌지

---

## 4. 레이아웃 구조

### 페이지 (단일 컬럼 문서, `.page` padding `26px 44px 38px`)
- 상단 여백 있는 편집형 레이아웃, 최소폭 1200px

### 주요 섹션 (세로 흐름)
1. **탑바** (`62px`): `320px 1fr auto` — 브랜드 / 중앙 stream / 검색·도움말·아바타
2. **네비** (`.nav`): 텍스트 탭(브리프/빌드/Perforce/WorkHub/릴리스/캘린더), active=오렌지 밑줄
3. **히어로** (`.hero`, `1.05 : 0.95`):
   - 좌: 아이브로우 + 대형 세리프 H1 + 설명 문단 + 메타
   - 우: 마일스톤 — 원형 장식(`:after`), D-38 거대 카운트, 진행률 바
4. **메인** (`.main`, `1.3 : 0.7`):
   - 좌: 프로덕션 타임라인 (세로 이벤트 리스트)
   - 우: 서비스 상태 카드 2×2 + 오늘의 회의 공지
5. **푸터**: 내부 표기 + 동기화 시각

---

## 5. 컴포넌트

### 히어로 마일스톤 (`.milestone`)
- 원형 오렌지 아웃라인 장식(`:after`, 260px, 우상단 오버플로우)
- `NEXT MILESTONE` + `ON TRACK`(올리브 발광 점)
- 거대 D-38 세리프 숫자 + 라벨
- 전체 진행률 68% 바(오렌지)

### 타임라인 (`.event`)
- `86px 20px 1fr auto` 그리드: 날짜 / 레일 / 내용 / 태그
- 세로 연결선(`.rail:after`), 원형 dot
- dot 상태: `.done`=올리브, `.now`=오렌지, 기본=페이퍼(예정)
- 태그: 알약형(`border-radius:20px`) 라운드 태그

### 서비스 상태 카드 (`.card`)
- 2×2 그리드, 상단 3px 컬러 바(올리브/`.warn` 오렌지)
- 라벨+상태 점 → 세리프 상태값(22px) → 세부
- hover: 떠오름 + 크림 배경

### 공지 (`.notice`)
- 원형 오렌지 날짜 뱃지(`TODAY 16:00`) + 회의 제목·설명

---

## 6. 반응형

- `@media(max-width:1300px)`:
  - H1 `58px → 48px`, D-day 카운트 `132px → 105px`
  - 페이지 좌우 패딩 `44px → 28px`
