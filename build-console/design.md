# Design Spec — Build Console (Option C)

UE5 개발 관제 대시보드. **터미널/CI 콘솔** 미학의 다크 테마 단일 화면 레이아웃.
출처: `option-c.html`

## 샘플 화면

![Build Console 샘플 화면](./option-c.png)

> 렌더링 원본: [`option-c.html`](./option-c.html)

---

## 1. 디자인 컨셉

- **테마**: 다크 모드 전용, 개발자 콘솔 / 빌드 파이프라인 관제 화면
- **분위기**: 라임 그린 액센트의 모노스페이스 터미널, 배경 그리드 오버레이
- **밀도**: 고밀도 정보 대시보드 (한 화면에 지표·파이프라인·체인지리스트·서비스 상태 집약)
- **최소 폭**: `1180px` (데스크톱 전용, `overflow:hidden`)

---

## 2. 컬러 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--bg` | `#0d0f0e` | 페이지 배경 (거의 검정) |
| `--panel` | `#141714` | 패널 배경 |
| `--panel2` | `#191c18` | 보조 패널 배경 |
| `--line` | `#30352e` | 경계선 / 구분선 |
| `--text` | `#e8ece4` | 기본 텍스트 |
| `--muted` | `#858d80` | 보조 텍스트 / 레이블 |
| `--lime` | `#b9ff38` | 주 액센트 (정상·활성·강조) |
| `--amber` | `#ffbd3e` | 경고 / 실행 중 / 카운트다운 |
| `--red` | `#ff625d` | 에러 (정의됨, 현재 미사용) |
| `--cyan` | `#63d7db` | 링크성 ID (체인지리스트 번호) |

**상태 색 규칙**: 정상=라임, 진행/경고=앰버, 대기=뮤트 그레이.

---

## 3. 타이포그래피

- **폰트 패밀리**: `"Cascadia Mono","SFMono-Regular",Consolas,"Malgun Gothic",monospace`
  - 전면 모노스페이스, 한글은 맑은 고딕 폴백
- **주요 사이즈 스케일**
  - H1 헤드라인: `28px` / `font-weight:500` / `letter-spacing:-.04em`
  - 카운트다운(D-38): `31px` / 앰버
  - 지표 수치(`.big`): `23px`
  - 패널 타이틀: `11px` / `letter-spacing:.08em`
  - 레이블(`.label`): `9px` / 대문자 / `letter-spacing:.14em`
  - 콘솔·테이블 본문: `9~10px`
- **아이브로우/레이블**: 넓은 자간 + 대문자 = 계기판 느낌

---

## 4. 레이아웃 구조

### 앱 골격 (2열 × 2행 그리드)
```
grid-template-columns: 76px 1fr    (레일 | 콘텐츠)
grid-template-rows:    64px 1fr    (탑바 | 메인)
```

- **좌측 레일** (`76px`, 세로 전체): 로고 마크 `SP` + 아이콘 네비 (OPS/BUILD/P4/WORK/LOG) + 하단 HELP. 활성 항목은 라임 좌측 마커.
- **탑바** (`64px`): 브레드크럼 · 스트림 배지 · 우측 시계/환경(PRODUCTION)/아바타
- **메인**: 3행 그리드 — `auto`(헤드라인) / `116px`(지표) / `1fr`(워크스페이스)
  - 배경에 `28px` 격자 그리드 오버레이 (미세한 흰색 라인)

### 워크스페이스 (2열)
```
grid-template-columns: minmax(0,1.62fr) minmax(360px,.82fr)
```
- **좌**: 액티브 빌드 파이프라인 패널
- **우 (right-stack, 2행)**: 최근 체인지리스트 + 서비스 상태

---

## 5. 컴포넌트

### 지표 바 (`.metrics`)
- 5개 셀 가로 배치, `grid-template-columns:1.15fr 1fr 1fr 1fr 1fr`
- 각 셀: 레이블(소·대문자) → 수치(대) → 서브텍스트
- 상태 점(`.status-dot`): 라임 글로우(`box-shadow:0 0 8px`), 앰버 변형
- 스파크라인(`.spark`): 5개 막대, 최고값 막대만 라임 강조

### 빌드 파이프라인 패널
- 헤드: 타이틀 + 탭(CURRENT/HISTORY/AGENTS), 활성 탭 라임 밑줄
- 서머리: 빌드 ID + 진행 바(`.progress`, 라임 채움 76%) + 퍼센트
- 스테이지 행(`.stage`): `30px 1.2fr 1fr 72px 70px` 그리드
  - 번호 · 이름 · 명령어(생략표시) · 시간 · 상태 태그
  - 태그 상태: `PASSED`(라임) / `RUNNING`(앰버 + 회전 로더) / `QUEUED`(그레이)
- 콘솔(`.console`, `72px`): 검정 배경 터미널 로그, `prompt`=라임 / `warn`=앰버

### 체인지리스트 테이블 (`.cl-row`)
- `72px 52px 1fr 52px` — CHANGE(시안) / OWNER / DESC(생략) / FILES(우정렬)
- 헤더 행 별도 배경

### 서비스 상태 (`.services`)
- 2×2 그리드, 각 셀: 서비스명 + 상태(● OPERATIONAL 라임 / DEGRADED 앰버) + 업타임

---

## 6. 인터랙션 · 모션

- **회전 로더**: `@keyframes spin` 1s linear infinite (실행 중 스테이지)
- **호버**: 레일 네비 hover 시 라임 텍스트 + 경계 강조
- 정적 목업 — 실동작 상태 전환 없음(값 하드코딩)

---

## 7. 반응형

- `@media(max-width:1250px)`:
  - 레일 `76px → 62px`
  - 지표 그리드 첫 열 비율 조정, 스파크라인 숨김
- 기본적으로 데스크톱 전용 (min-width 1180px, body overflow hidden)

---

## 8. 재현 체크리스트

- [ ] 모노스페이스 폰트 스택 + 다크 배경 `#0d0f0e`
- [ ] 라임(`#b9ff38`) 단일 액센트, 앰버=경고, 시안=ID
- [ ] 좌측 아이콘 레일 + 탑바 + 격자 배경 메인
- [ ] 5셀 지표 바 (상태점·스파크라인)
- [ ] 빌드 파이프라인(진행바·스테이지 태그·회전 로더·터미널 콘솔)
- [ ] 우측 스택: 체인지리스트 테이블 + 서비스 상태 2×2
- [ ] 넓은 자간 대문자 레이블로 계기판 톤 유지
