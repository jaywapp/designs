## 1. 디자인 컨셉

- **테마**: 라이트 모드, 스포츠 신문/매거진 편집 디자인
- **분위기**: 초대형 압축 헤드라인 + 세로 규칙선(column rule) + 반듯한 각진 그림자
- **밀도**: 히어로 중심의 대담한 편집 레이아웃 + 하단 대시보드 스트립
- **최소 폭**: `320px` (반응형 대응)
- **접근성**: `word-break:keep-all`(한글 어절 보존), `focus-visible` 노란 아웃라인, aria-label 다수

## 3. 타이포그래피

- **폰트**: `"Arial Narrow","Pretendard","Noto Sans KR",Arial,sans-serif` — **압축형(narrow)** 산세리프가 정체성
- **헤드라인 H1**: `clamp(76px,7.1vw,116px)` / `font-weight:950` / `line-height:.9` / `letter-spacing:-7px` — 초대형·초타이트
- **라벨/아이브로우**: Arial, `900` 볼드, 넓은 자간(`letter-spacing:2~2.7px`), 대문자
- **본문 lead**: `17px` / `line-height:1.75`
- **숫자 강조**: 날짜 `950 34px`, 스코어 `950 44px` — 스포츠 결과지 느낌

---

## 4. 레이아웃 구조

### 페이지 (`.page`, 세로 규칙선 배경)
- 좌측 6.5% 지점에 세로 컬럼 규칙선(linear-gradient) — 신문 그리드 암시

### 섹션 흐름
1. **탑바** (`100px`): `360px 1fr auto` — 크레스트+브랜드 / 중앙 네비 / 회원 로그인. 활성 네비 그린 언더라인
2. **히어로** (`.hero`, `57% : 43%`):
   - 좌(페이퍼): 이슈 라벨 → 초대형 H1(그린 강조어) → 시즌+lead 2열 → CTA(각진 그림자)
   - 우(네이비): 거대 배경 숫자 "07", NEXT FIXTURE + **티켓 카드**(양쪽 반원 노치·점선) + 최근 결과 스코어
3. **대시보드 스트립** (`.dashboard-strip`, `1.2 : 0.8 : 0.75`): 멤버 라커룸 미리보기 — RSVP / 공지 / 회비 3패널

## 5. 컴포넌트

### 크레스트 (`.crest`)
- 방패형(하단 라운드 `6px 6px 22px 22px`), 라임 배경 + 네이비 각진 그림자, `::after`로 "2018" 각인

### 티켓 카드 (`.ticket`)
- 실물 입장권 메타포: 양측 반원 노치(`::before/::after` 네이비 원), 점선 구분선(`dashed`)
- 헤드(매치넘버+날짜) → 대진(`1fr auto 1fr`, VS 그린 이탤릭) → 경기장(venue)
- 두꺼운 그린 각진 그림자 `14px 14px 0`

### CTA (`.cta`)
- primary: 네이비 배경 + 그린 각진 그림자
- secondary: 밑줄 텍스트 버튼

### 대시보드 패널 (`.panel`)
- 상단 4px 컬러 바(네이비 / `.green-top` 그린), 소프트 드롭섀도
- RSVP 버튼(참석 활성=그린), 회비 금액 `950 28px` + "납부 완료" 뱃지

---

## 6. 반응형

- `@media(max-width:900px)`:
  - 네비 숨김, 히어로/대시보드 1열
  - H1 `116px → 68px`, lead-row 1열, 스트립 헤드 세로 정렬

---

## 7. 재현 체크리스트

- [ ] 웜 페이퍼 배경 + 좌측 세로 컬럼 규칙선
- [ ] Arial Narrow 초대형 압축 헤드라인(그린 강조어)이 핵심
- [ ] 네이비 다크 사이드 + 거대 배경 숫자
- [ ] 실물 입장권 스타일 티켓 카드(반원 노치·점선·각진 그림자)
- [ ] 각진 오프셋 그림자(`Npx Npx 0`) 반복 사용
- [ ] 하단 3패널 멤버 라커룸(RSVP/공지/회비)
- [ ] 한글 keep-all + focus-visible 접근성
